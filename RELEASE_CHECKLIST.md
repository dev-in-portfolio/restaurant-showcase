# Showcase Release Checklist

Before adding a restaurant to the showcase:

- [ ] All review flags in `restaurant.json` are `true`
- [ ] `status` is set to `"approved"`
- [ ] No placeholder text exists in any HTML file
- [ ] All local images, scripts, and stylesheets resolve
- [ ] `index.html` exists and loads correctly
- [ ] No temporary or development files present
- [ ] Comparison button is added or explicitly marked N/A
- [ ] Production build passed
- [ ] Restaurant passes showcase validation
- [ ] Showroom index entry is created and valid
- [ ] No duplicate slugs in the index
